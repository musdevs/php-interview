## Настройка прокси для просмотра заблокированных сайтов

[Прокси для ленивых: поднимаем SOCKS5 поверх SSH, пока чайник закипает](https://habr.com/ru/articles/974040/)

```shell
ssh -p 21589 -D 1081 -N -C user@example.com
```

Где:

-D 1081: поднимает локальный SOCKS5 на порту 1081.
-N: говорит SSH «не открывай мне шелл, я тут только ради туннеля».
-C: включает сжатие (полезно, если вы читаете текст, бесполезно, если смотрите 4K-видео).

В Firefox настраиваем SOCKS v5? HTTP Proxy и HTTPS Proxy должны быть пустые:

SOCKS Host: 127.0.0.1
Port: 1081
SOCKS v5

## Открыть прокси на заданном интерфейсе

```shell
ssh -p 21589 -D 192.168.88.50:1081 -N -C user@example.com
```
