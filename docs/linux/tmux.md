# tmux

## Сессии

### Создать сессию 0

```shell
tmux
```

### новая сессия session1
```shell
tmux new -s session1
```

### Подключение к сессии, либо к единственной, либо последней созданной

```shell
tmux attach
```

### Подключение к сессии session1

```shell
tmux attach -t session1
```

## Окна

### Новое окно (нажать CTRL+b, затем нажать с)
<C-b c>

### Список окон
<C-b w> // переключиться курсором вверх-вниз

### Переключение
<C-b n> // следующее окно
<C-b p> // предыдущее окно
<C-b 0> // переключиться на номер окна
