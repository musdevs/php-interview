# VS Code short keys

## Выделение области

Последовательно нажимаем Shift+Alt+Right или Shift+Alt+Left и выделяется
сначала слово, потом строка, потом абзац.

## Выделение всех слов

1. Дважды кликнуть на нужном слове
2. Shift+Alt+L - выделить все слова

Появится мультикурсор. Ненужные выделения можно снять Alt+mouse click

## Переход к парному тегу

Ctrl+M

## Свернуть содержимое тега

Ctrl + Shift + [

## Развернуть содержимое тега

Ctrl + Shift + [
