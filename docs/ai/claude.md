## [Everything Claude Code](https://github.com/affaan-m/everything-claude-code)

👁 Это большой набор скиллов, скриптов и конфигураций, который превращает Claude Code в полноценного AI-агента для разработки.

Содержит готовые команды для code review, написания тестов, миграций баз данных и других задач, а также систему с памятью, правилами, хуками и MCP-конфигурациями. Работает не только с Claude Code, но и с другими AI-агентами, позволяя автоматизировать разработку на уровне целых процессов.

❗️  Отлично подойдёт тем, кто хочет прокачать AI-агентов и превратить терминал в полноценный инструмент автопилота для разработки.
