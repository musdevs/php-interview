# Astra

## Версия

```shell
$ cat /etc/astra_version
1.7.7
```

## Лицензия

```shell
$ cat /etc/astra_license
MODE=1
DESCRIPTION=advanced(voronezh)
URL=https://astralinux.ru/information/licenses
```

## Версия

```shell
~$ cat /etc/*-release
DISTRIB_ID="AstraLinux"
DISTRIB_DESCRIPTION="Astra Linux 1.7 x86-64"
DISTRIB_RELEASE=1.7_x86-64
DISTRIB_CODENAME=1.7_x86-64
PRETTY_NAME="Astra Linux"
NAME="Astra Linux"
ID=astra
ID_LIKE=debian
ANSI_COLOR="1;31"
HOME_URL="https://astralinux.ru"
SUPPORT_URL="https://astralinux.ru/support"
LOGO=astra
VERSION_ID=1.7_x86-64
VERSION_CODENAME=1.7_x86-64
```

## Релиз

```shell
$ lsb_release -a
No LSB modules are available.
Distributor ID: AstraLinux
Description:    Astra Linux 1.7 x86-64
Release:        1.7_x86-64
Codename:       1.7_x86-64
```

Codename: 1.7_x86-64	Это и есть кодовое имя для вашей версии. Оно используется в командах apt
