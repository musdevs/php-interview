# sudo

## Выполнять sudo без пароля

Редактировать sudoers напрямую небезопасно!
Любая ошибка в синтаксисе файла приведет к потере прав суперпользователя!

visudo открывает файл sudoers в безопасном режиме:
- не дает сохранить файл с ошибкой
- блокирует от изменения файла другими пользователями или процессами
- запускает проверенный редактор

```shell
visudo /etc/sudoers
ivan ALL=(ALL) NOPASSWD:ALL
```
