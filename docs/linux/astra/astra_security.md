## Поднять уровень до максимального

Поднять уровень можно только из той сессии, у которой уровень это позволяет делать

```shell
sudo pdpl-user -i 63 vvs
minimal pdpl: Уровень_0:Low:Нет:0x0
   0:0:0x0:0x0
maximal pdpl: Уровень_0:High:Нет:0x0
   0:63:0x0:0x0
deploy@portal-wfm-dev:~$ pdp-id
Level=0(Уровень_0), ILevel=63(High), Categories=0x0(Нет)
Roles=()

```
