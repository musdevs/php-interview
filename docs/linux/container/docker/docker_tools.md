# Docker tools

## OCI2Git

👁 Это инструмент, который преобразует Docker-образы в Git-репозитории, сохраняя структуру и историю слоёв.

Каждый слой контейнера превращается в отдельный коммит, что позволяет анализировать изменения и состав образа как обычный Git-проект. Дополнительно генерируется fsbom в YAML с перечнем файлов по слоям и извлекаются метаданные в удобном формате.

❗️ Отлично подойдёт тем, кто хочет глубже анализировать контейнеры и понимать, что именно внутри Docker-образов.
