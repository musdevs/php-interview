# Игнорировать ошибки прав доступа в WinSCP

Откройте WinSCP и в главном меню перейдите в Options (Настройки) > Options (Параметры) .

В открывшемся окне слева выберите раздел Transfer (Передача).

Нажмите кнопку Edit... (Редактировать...) в блоке "Transfer settings" (Настройки передачи) .

В появившемся окне найдите и отметьте галочкой пункт Ignore permission errors (Игнорировать ошибки прав доступа) .

Нажмите OK во всех окнах, чтобы сохранить настройки.
